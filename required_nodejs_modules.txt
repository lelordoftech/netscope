This project requires node.js and the following modules to be installed for successful compilation
install with npm install <package>[@<version>]

tablesorter@2.26
tableify
coffeeify
coffeescript
watchify (globall)

#####
curl -sL https://deb.nodesource.com/setup_11.x | sudo -E bash -
sudo apt-get install -y nodejs

sudo npm install -g watchify
sudo npm install tablesorter@2.26 tableify coffeeify coffeescript

#####
Example:
Ubuntu 16.04 LTS

nodejs v11.2.0
node v11.2.0
npm 6.4.1
